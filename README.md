## Getting Started

Clone the repository:

```bash
git clone https://github.com/cryptic0053/Portfolio-n.git
cd Portfolio-n
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```bash
http://localhost:3000
```

## Production Build

To build the project for production:

```bash
npm run build
```

To start the production server:

```bash
npm run start
```
